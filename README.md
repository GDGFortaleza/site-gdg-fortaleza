site-gdg-fortaleza
==================

Site GDG Fortaleza baseado no GDG Boilerplate

Instalar Dependencias para desenvolver o projeto
==================================
1.instalar o nodejs.
	verificar se foi instalado o gerenciamento de package npm.

2.instalar o bower 
	npm install bower -g 


Instalar Dependencias Web
==================================
bower install
